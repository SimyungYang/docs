이 워크샵에서는 Databricks 플랫폼의 AI 기능을 활용해 LG Smart TV(Genie 플랫폼)에 탑재되는 AI 기반 콘텐츠 추천 및 음성 인터페이스 기능을 처음부터 직접 구현합니다. 코드를 작성하는 내내 AI가 함께하는 **AI Vibe Coding** 방식으로 진행되며, Databricks Workspace 안에서 모델 서빙·피처 엔지니어링·API 연동을 한 번에 경험할 수 있습니다.

## 이 워크샵에서 배우는 것

이 워크샵은 단순한 튜토리얼이 아닙니다. 실제 SmartTV 서비스와 비슷한 시나리오를 바탕으로 Databricks AI 기능을 처음부터 끝까지 직접 경험합니다.

<Info>
  워크샵 GitHub 리포지터리: [SimyungYang/genie-code-workshop-smarttv](https://github.com/SimyungYang/genie-code-workshop-smarttv)
</Info>

### 학습 목표

- Databricks Workspace에 접속하고 클러스터를 생성·관리합니다.
- Delta Lake 테이블에 시청 이력 데이터를 적재하고 피처로 변환합니다.
- Databricks Model Serving 엔드포인트를 배포하고 REST API로 호출합니다.
- LLM(대규모 언어 모델)을 활용해 자연어 기반 콘텐츠 추천 로직을 구현합니다.
- MLflow로 실험을 추적하고 모델 버전을 관리합니다.
- SmartTV Genie SDK와 Databricks 엔드포인트를 연동합니다.

### 구현하는 기능

| 모듈 | 기능 | 핵심 Databricks 서비스 |
|------|------|----------------------|
| 1 | 시청 이력 기반 콘텐츠 추천 | Delta Lake, Feature Store |
| 2 | LLM 기반 자연어 검색 | Model Serving, DBRX / LLaMA 3 |
| 3 | SmartTV Genie 연동 API | REST API, Model Serving Endpoint |

## 워크샵 흐름

<Steps>
  <Step title="환경 설정">
    Databricks 계정을 확인하고 개발 환경을 구성합니다. 워크샵용 클러스터와 카탈로그를 생성합니다.
  </Step>
  <Step title="데이터 준비">
    샘플 시청 이력 데이터셋을 Delta Lake 테이블에 적재하고, 추천 모델에 필요한 피처를 생성합니다.
  </Step>
  <Step title="AI 모델 개발">
    LLM 기반 추천 로직을 작성하고 MLflow로 실험을 추적합니다. 최적 모델을 Model Registry에 등록합니다.
  </Step>
  <Step title="API 배포 및 연동">
    Model Serving 엔드포인트를 배포하고 Genie SDK에서 REST 호출로 연동합니다.
  </Step>
  <Step title="마무리 및 다음 단계">
    완성된 기능을 검토하고 프로덕션 확장을 위한 다음 학습 경로를 확인합니다.
  </Step>
</Steps>
